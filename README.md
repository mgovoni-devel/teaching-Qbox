# teaching-Qbox
Introduction to the Qbox code (qboxcode.org)

The notebook shows how to use Qbox for the calculation of the electronic structure of C<sub>60</sub>.<br>

### Dependencies
* Python 3 
* Jupyter Notebook
* Numpy
* Matplotlib 

### Run the notebook
```sh
$ jupyter-notebook Qbox-C60.ipynb
```
